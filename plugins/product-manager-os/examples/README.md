# Worked Examples — Cadence

This folder is the OS, *shown working*. Every skill was run once against a single demo company so you can see exactly what it produces before you install.

- **[SAMPLE-OUTPUTS.md](SAMPLE-OUTPUTS.md)** — all 40 outputs in one document, with a categorized table of contents. Start here.
- **`cadence/<skill>.md`** — the individual output from each skill.

**The demo company — Cadence:** a B2B SaaS (~$3M ARR, ~4,000 teams) that replaces synchronous standups with async written check-ins + an AI digest. Current priorities: fix week-2 retention, launch an "AI Blocker Detector." All numbers, quotes, and names in these examples are fabricated for illustration.

**House style applied:** the outputs were generated with a defined house style (TL;DR-led, sentence-case headings, "teams" not "customers", 🟢🟡🔴 status markers) — to show how the OS conforms to a company's format. Set your own in `memory/house-style.md` or via the `house-style` skill, and your outputs come out in *your* format.
